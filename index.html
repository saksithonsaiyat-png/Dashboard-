คู่มือการใช้งาน & การเชื่อมต่อหลังบ้าน

1. การทำงานของระบบ (How it works)

  • ระบบนี้สามารถ อัปโหลดไฟล์ Excel / CSV เข้ามาเพื่อคำนวณ Progress ของงานแบบอัตโนมัติ
  • Routine (งานประจำ): ระบบจะกรองจากวันที่ (Plan Date) ถ้าระบุว่า `required = false` (หรือ No) งานนั้นจะไม่แสดงในวันนั้น
  • Adhoc (งานโปรเจค/สั่งการ): ระบบจะติดตามจาก `deadline` และคำนวณสถานะ Overdue (เลยกำหนด) หรือ Warning (ใกล้ถึงกำหนด 1-2 วัน)
  • เมื่อแก้ไข Progress (1-100%) หรือติ๊กถูกว่าเสร็จแล้วบนหน้าเว็บ ระบบจะปรับสถานะให้และกราฟจะอัปเดตทันที

2. วิธีตั้งชื่อหัวคอลัมน์ใน Excel (จำเป็น)

เพื่อให้สคริปต์อ่านค่าได้ถูกต้อง แถวแรกของ Excel ต้องมีชื่อคอลัมน์ดังนี้ (ตัวพิมพ์เล็กหรือใหญ่ก็ได้):

ชื่อคอลัมน์ความหมาย / ตัวอย่างข้อมูล
categoryใส่คำว่า Routine หรือ Adhoc
planDateวันที่ต้องทำ (สำหรับ Routine) รูปแบบ YYYY-MM-DD เช่น 2026-04-01
jobNameชื่อกลุ่มงาน เช่น Finance, IT Support
taskNameชื่องาน (หรือใช้คอลัมน์ title สำหรับ Adhoc ก็ได้)
requiredเฉพาะ Routine วันนี้ต้องทำไหม? ใส่ Yes หรือ No
ownerNameชื่อผู้รับผิดชอบงาน เช่น สมชาย ใจดี
deadlineกำหนดส่งงาน (สำหรับ Adhoc) รูปแบบ YYYY-MM-DD
progressความคืบหน้า 0 ถึง 100 (ใส่เป็นตัวเลข)
workStatusสถานะงาน not_started, in_progress, done

3. สคริปต์เชื่อมต่อหลังบ้าน (Excel 365 Office Script)

หากลูกค้าต้องการนำ Dashboard นี้ไปฝังใน Sharepoint หรือต่อ API อัตโนมัติ สามารถใช้ Power Automate ร่วมกับ Office Scripts ด้านล่างนี้ โดยนำไปสร้างในเมนู Automate บน Excel 365

getTasksData.ts (Office Script) 
function main(workbook: ExcelScript.Workbook): string {
  // 1. ดึง Sheet ที่ชื่อ "Tasks" หรือดึง Sheet แรก
  let sheet = workbook.getWorksheet("Tasks") || workbook.getWorksheets()[0];
  
  // 2. ดึง Table ตัวแรกใน Sheet
  let table = sheet.getTables()[0];
  if (!table) return JSON.stringify({ error: "No table found in the sheet" });

  // 3. อ่านหัวคอลัมน์ และ ข้อมูล
  let headerRange = table.getHeaderRowRange().getValues()[0];
  let dataRange = table.getRangeBetweenHeaderAndTotal().getValues();

  let tasks = [];

  // 4. แปลงข้อมูลแถวให้เป็น JSON Object
  for (let i = 0; i < dataRange.length; i++) {
    let row = dataRange[i];
    let taskObj: { [key: string]: any } = {};
    
    for (let j = 0; j < headerRange.length; j++) {
      let headerName = headerRange[j].toString().trim();
      let value = row[j];
      
      // แปลงวันที่ Excel Serial (ถ้ามี) เป็น YYYY-MM-DD
      if (typeof value === "number" && (headerName.toLowerCase().includes("date") || headerName.toLowerCase() === "deadline")) {
        let date = new Date(Math.round((value - 25569) * 86400 * 1000));
        taskObj[headerName] = date.toISOString().split("T")[0];
      } else {
        taskObj[headerName] = value;
      }
    }
    // เพิ่มเฉพาะแถวที่มีข้อมูล
    if(taskObj["taskName"] || taskObj["title"]) {
      tasks.push(taskObj);
    }
  }

  // ส่งค่ากลับเป็น JSON String เพื่อให้ Power Automate หรือ API นำไปใช้ต่อ
  return JSON.stringify(tasks);
}

ไอเดียการต่อยอด: ใช้ Power Automate ตั้งเวลา (Scheduled Cloud Flow) ให้รันสคริปต์นี้ทุกๆ 1 ชั่วโมง แล้วส่งผลลัพธ์ JSON ไปอัปเดตไฟล์บนเซิร์ฟเวอร์ หรือส่งเข้า REST API เพื่อให้หน้า Dashboard แสดงผล Real-time
