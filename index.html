เริ่มต้นใช้งาน + เชื่อม Excel

วิธีใช้งานเว็บไซต์เบื้องต้น

  1. เลือกวันที่จาก Date Picker เพื่อดูงานประจำวัน
  2. กด Import Excel หรือดาวน์โหลด CSV ตัวอย่างไปกรอกก่อนนำเข้า
  3. Routine แสดงแบบแยก Job และกดขยายดู task รายข้อได้
  4. ปรับ Progress / สถานะงาน ได้ทันทีในหน้า Dashboard (บันทึกลง localStorage)
  5. หากมี Backend แล้ว กด Sync Excel 365 เพื่อดึงข้อมูลล่าสุดจาก Microsoft Graph

สถานะเชื่อมต่อ: ยังไม่เชื่อม Backend (โหมด Local Import)

วิธีจัดคอลัมน์ Excel ให้สคริปต์อ่านได้ + ตัวอย่าง

คอลัมน์ที่รองรับ: category, planDate, jobName, taskName, title, required, ownerId, ownerName, jobType, receiveDate, deadline, progress, workStatus, actualDelivery

ตัวอย่าง Routine: category=Routine, planDate=2026-04-01, jobName=Finance, taskName=กระทบยอดเงินสด, required=Yes

ตัวอย่าง Adhoc: category=Adhoc, title=เตรียมเอกสารประชุมบอร์ด, receiveDate=2026-04-01, deadline=2026-04-03

หมายเหตุ: ถ้าวันไหนไม่ต้องทำ Routine task ให้ระบุ required=No แล้วระบบจะไม่แสดง task ในวันนั้น

วิธีเชื่อมหลังบ้านกับ Excel 365 (Microsoft Graph)

1) ตั้งค่า Azure App Registration + สิทธิ์ Files.Read.All / Sites.Read.All (Application permissions)

2) ใช้ไฟล์ excel365-backend-example.js เป็นตัวอย่าง backend endpoint `/api/excel/tasks`

3) ตั้งค่า URL backend ที่ตัวแปร graphConfig.endpoint ในไฟล์นี้ แล้วกดปุ่ม Sync Excel 365

4) backend ต้อง map คอลัมน์จาก Excel ให้เป็น keys เดียวกับ parser ด้านหน้า

* ตัวอย่าง backend script ถูกเพิ่มไว้ในโปรเจกต์แล้ว: excel365-backend-example.js
